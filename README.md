# [StreamWheel](https://streamwheel.github.io/)
StreamWheel is a nice solution for creating wheels that you can spin on livestreams using OBS. It's extremely customizable, easy to use, self-hostable, and with no dependencies. You can add images to wheels, customize colors, customize how it behaves, add sounds, and more. When you're done, you can export it into an .html file that is entirely self-contained, it contains all of the images and sounds that you added, so the resulting .html file is extremely portable as you don't have to also move all of your assets around. You can then import your imported .html file in order to edit it.

Once you export your .html, import it into OBS as a browser source. You can go from there.

Enjoy! :)
